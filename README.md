# 🚀 AI Resume Screening System

## 📌 Overview
The **AI Resume Screening System** is a Java-based web application designed to automate the recruitment process by analyzing and comparing candidate resumes with job descriptions.
It uses **AI-inspired keyword matching techniques** to evaluate resumes, calculate matching scores, and rank candidates efficiently, reducing manual effort and improving hiring accuracy.

## 🎯 Objectives
* Automate resume screening process
* Reduce recruiter workload and time
* Minimize human bias in candidate selection
* Provide structured and efficient candidate ranking

## 🛠️ Tech Stack
| Layer        | Technology Used |
| ------------ | --------------- |
| Frontend     | HTML, CSS, JSP  |
| Backend      | Java Servlets   |
| Database     | MySQL           |
| Server       | Apache Tomcat   |
| Connectivity | JDBC            |

## ⚙️ Key Features
* 📄 Resume Upload & Processing
* 🔍 Keyword Extraction from Resumes
* 📊 Matching with Job Description
* 📈 Candidate Scoring System
* 🧑‍💼 Admin Dashboard
* 📂 Candidate & Job Management

## 🧠 Working Principle
1. Extract keywords from candidate resumes
2. Extract required skills from job description
3. Compare both datasets
4. Calculate matching percentage
5. Rank candidates based on score

## 🏗️ System Architecture
User (Candidate/Admin)
        ↓
Frontend (HTML, CSS, JSP)
        ↓
Backend (Java Servlets)
        ↓
Database (MySQL)

## 🚀 How to Run the Project
1. Install **Apache Tomcat Server**
2. Configure **MySQL Database**
3. Import project into **Eclipse IDE**
4. Add required libraries (JDBC, PDFBox)
5. Deploy project on Tomcat server
6. Run application in browser
